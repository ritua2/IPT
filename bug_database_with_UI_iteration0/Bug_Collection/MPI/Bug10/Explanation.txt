MPI_Type_get_extent arguments cannot be NULL.

In details, in the buggy version:

``````````````````````````````
   MPI_Type_get_extent(MPI_A, NULL, &extent);
``````````````````````````````

This will cause the program to crash.


* Error

```
srun: error: nid00031: tasks 0-3: Segmentation fault
```
